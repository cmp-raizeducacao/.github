## Segurança

Informações confidenciais, dados pessoais, credenciais e configurações internas não devem ser expostos em repositórios públicos.

Caso identifique qualquer risco de segurança, comunique o responsável pelo projeto ou a coordenação antes de registrar uma issue pública.
